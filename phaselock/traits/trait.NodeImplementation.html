Expand description

Node implementation aggregate trait

This trait exists to collect multiple behavior implementations into one type, to allow PhaseLock to avoid annoying numbers of type arguments and type patching.

It is recommended you implement this trait on a zero sized type, as PhaseLockdoes not actually store or keep a reference to any value implementing this trait.
