Module phaselock::demos::dentry[−][src]

Expand description

Double entry accounting demo

This module provides an implementation of the PhaseLock suite of traits that implements a basic demonstration of double entry accounting.

These implementations are useful in examples and integration testing, but are not suitable for production use.
