pub struct NetworkDef {
    gossipsub: Behaviour,
    pub dht: Behaviour<MemoryStore>,
    identify: Behaviour,
    pub direct_message: Behaviour<Vec<u8>, Vec<u8>>,
    pub request_response: Behaviour<Request, Response>,
    pub autonat: Behaviour,
}
Expand description

Overarching network behaviour performing:

  • network topology discovoery
  • direct messaging
  • p2p broadcast
  • connection management

Fields§

§gossipsub: Behaviour

purpose: broadcasting messages to many peers NOTE gossipsub works ONLY for sharing messages right now in the future it may be able to do peer discovery and routing https://github.com/libp2p/rust-libp2p/issues/2398

§dht: Behaviour<MemoryStore>

purpose: peer routing purpose: storing pub key <-> peer id bijection

§identify: Behaviour

purpose: identifying the addresses from an outside POV

§direct_message: Behaviour<Vec<u8>, Vec<u8>>

purpose: directly messaging peer

§request_response: Behaviour<Request, Response>

Behaviour for requesting and receiving data

§autonat: Behaviour

Auto NAT behaviour to determine if we are publically reachable and by which address
