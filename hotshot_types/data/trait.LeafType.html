fn fill_deltas( &mut self, block: LeafBlock<Self> ) -> Result<(), LeafDeltasError<Self>>

Fill this leaf with the entire corresponding block.

After this function succeeds, self.get_deltas().try_resolve() is guaranteed to return Ok(block).

Errors

Fails if block does not match self.get_deltas_commitment(), or if the block is not able to be stored for some implementation-defined reason.
