    S: WebInfo + Send + 'static + Clone,
Expand description

Spawn a web server on the given addr. This web server will host the static HTML page /web/index.html and expose a sse endpoint. This sse endpoint will send status updates to the connected clients whenever NetworkNodeHandle::state_changed triggers.
